Load Balancer

A load balancer is a device that acts as a reverse proxy and distributes network or application traffic across a number of what is a load balancer servers.

Load Balancing Algorithms

  • Round Robin
  • Least Connections
  • Least Response Time
  • IP Hash

API Gateway

Problems

  • Different clients need different data.
  • Client could make that many requests over a LAN.
  • Some services might use protocols that are not web-friendly.
  • Service instances and their locations (host+port) changes dynamically.
  • Difficult to refactor the microservices.
